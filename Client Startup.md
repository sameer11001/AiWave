## runing

1. Make sure flutter is active by typing in the bash `cmd`

```bash
flutter --version
```

2. Enter the folder `aiwave`

3. Run the bash in the folder

4. Typein the bash

```bash
flutter pub get
```

5. After run pub get Type in bash

```bash
flutter run
```
